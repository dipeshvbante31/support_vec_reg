# 📈 Support Vector Regression (SVR) - Machine Learning

A complete implementation of **Support Vector Regression (SVR)** using Python and Scikit-learn. This project demonstrates how to preprocess data, handle categorical variables, train an SVR model, and evaluate regression performance using multiple evaluation metrics.

---

## 📌 Project Overview

This project focuses on building a **Support Vector Regression (SVR)** model to predict continuous values using the popular **Tips Dataset**.

The notebook covers the complete machine learning workflow including:

- Data Loading
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Handling Categorical Variables
- Feature Engineering
- Feature Scaling
- Training an SVR Model
- Model Evaluation

The project uses the **Tips Dataset**, which contains restaurant billing information.

## 📊 Evaluation Metrics

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score (Coefficient of Determination)

---

## 📈 Why Feature Scaling is Important?

Support Vector Regression is highly sensitive to feature magnitudes. Therefore, **StandardScaler** is applied before training to improve model performance and convergence.
